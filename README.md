## 🎯 游戏简介

这是一款以 **BanG Dream!** 系列为主题的猜杯子小游戏。玩家需要追踪户山香澄（KSM）躲藏的杯子位置，通过观察洗牌过程来猜测最终位置。游戏融合了经济系统、音乐播放器等丰富功能，提供沉浸式的游戏体验。

**核心玩法**：观察 → 追踪 → 投注 → 选择 → 获胜！

***

## ✨ 功能特性

### 核心玩法

| 功能         | 描述                                                                          |
| ---------- | --------------------------------------------------------------------------- |
| **猜杯子机制**  | 经典的三杯猜球游戏，追踪目标杯子的位置变化                                                       |
| **五档难度**   | Easy (3次) / Normal (6次) / Hard (10次) / Expert (15次) / Special (自定义 15-50 次) |
| **随机杯子外观** | 每次洗牌时杯子会随机变换外观，增加辨识难度                                                       |
| **随机小球角色** | 每局游戏小球会随机选择不同的 BanG Dream! 角色                                               |
| **实时状态追踪** | 显示当前连胜、最大连胜、胜负统计等数据                                                         |
| **杯子交换动画** | 优化的杯子交换动画，增加停留时间，提升视觉效果                                                     |

### 星石经济系统

游戏内置完整的经济系统，让游戏更具策略性：

```
初始星石: 100 颗
投注范围: 10 ~ 当前持有星石
收益公式: 猜对获得 (投入星石 × 洗牌次数) 颗星石
惩罚机制: 猜错扣除投入的星石
```

**收益示例**：

- Easy 难度投注 10 星石 → 猜对获得 30 星石
- Expert 难度投注 50 星石 → 猜对获得 750 星石

### 银行借贷系统

当星石不足时，可以向银行借款继续游戏：

| 借款金额区间     | 利息率 | 还款示例          |
| ---------- | --- | ------------- |
| ≤ 50       | 10% | 借 50 还 55     |
| 51 - 100   | 15% | 借 100 还 115   |
| 101 - 200  | 20% | 借 200 还 240   |
| 201 - 300  | 25% | 借 300 还 375   |
| 301 - 500  | 30% | 借 500 还 650   |
| 501 - 1000 | 35% | 借 1000 还 1350 |
| > 1000     | 40% | 借 2000 还 2800 |

**借款限制**：可借金额范围为 `10 ~ 当前星石 × 2`

### 表情包系统

游戏内置 **33 个** BanG Dream! 角色表情包：

- **点击发送**：点击表情包即可发送到屏幕随机位置

- **动画效果**：表情包出现时有放大旋转动画，3 秒后淡出消失
- **随机位置**：表情包会出现在屏幕任意位置，避免重叠

## 🚀 快速开始

### 方式一：直接打开（推荐）

1. 下载或克隆本项目到本地
2. 双击 `index.html` 文件
3. 在浏览器中自动打开游戏

### 方式二：本地服务器

使用任意本地服务器运行：

```bash
# Python 3
python -m http.server 8080

# Python 2
python -m SimpleHTTPServer 8080

# Node.js (需要安装 serve)
npx serve . -l 8080

# PHP
php -S localhost:8080
```

然后访问 `http://localhost:8080`

### 方式三：VS Code Live Server

1. 安装 VS Code Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 🎮 游戏操作指南

### 基本流程

```
1. 选择难度 → 2. 开始游戏 → 3. 投入星石
      ↓              ↓              ↓
4. 观察小球 → 5. 追踪洗牌 → 6. 选择杯子
      ↓              ↓              ↓
7. 查看结果 → 8. 获得奖励 → 9. 继续挑战
```

### 详细操作

| 步骤 | 操作   | 说明                        |
| -- | ---- | ------------------------- |
| 1  | 选择难度 | 点击难度按钮选择洗牌次数，Special 可自定义 |
| 2  | 开始游戏 | 点击"开始游戏"按钮                |
| 3  | 投入星石 | 在弹窗中选择或输入投入的星石数量          |
| 4  | 观察小球 | 记住小球所在的杯子位置（粉色小球）         |
| 5  | 追踪洗牌 | 仔细观察杯子的交换过程               |
| 6  | 选择杯子 | 点击你认为藏有小球的杯子              |
| 7  | 查看结果 | 猜对获得星石奖励，猜错损失投入的星石        |

### 快捷操作

- **设置面板**：点击右上角齿轮图标
- **音乐播放器**：点击右下角音乐图标
- **表情包面板**：点击表情包按钮
- **银行功能**：点击星石旁边的银行图标
- **商店面板**：点击左下角商店图标

### 商店操作

1. **打开商店**：点击左下角的商店图标
2. **浏览卡片**：查看商店中的卡片列表
3. **购买卡片**：点击卡片下方的"购买"按钮（需要100星石）
4. **放置卡片**：购买后卡片会自动放置到屏幕中央
5. **拖动卡片**：点击并拖动卡片到任意位置
6. **调整大小**：通过拖动卡片边缘调整大小
7. **关闭商店**：点击商店图标或外部区域关闭商店面板

### 注意事项

1. **音频自动播放限制**：首次打开游戏需要点击屏幕解锁音频播放权限（浏览器安全策略）
2. **数据持久化**：游戏数据保存在浏览器内存中，刷新页面会重置所有数据
3. **最佳体验**：建议使用最新版本的 Chrome 或 Firefox 浏览器
4. **网络要求**：首次加载需要从 CDN 加载 Vue.js 和字体文件

## 📊 性能优化

- **杯子图片预加载**：采用 JavaScript 预加载策略，在游戏初始化时预加载所有杯子图片，避免切换杯子时的加载延迟
- **图片懒加载**：表情包图片按需加载
- **音频预加载**：核心音效在游戏初始化时预加载
- **CSS 动画**：使用 CSS3 硬件加速动画
- **事件节流**：鼠标移动事件进行节流处理
- **内存管理**：及时清理 DOM 元素和音频资源
